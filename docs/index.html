Recipe & Store Info Extractor

スクショ3枚から、レシピと店舗情報を整理する。

この画面ひとつで、画像の読み込み、OCR、LLMによるJSON化、IndexedDBへの保存、タグ検索まで進められます。 モバイルでも読みやすいように、コントラストと余白を強めに整えています。

🧠 小さいモデルを自動取得

Transformers.js 対応の Hugging Face モデルIDを指定すると、ブラウザが自動でダウンロードします。

URL: https://huggingface.co/onnx-community/Qwen2.5-0.5B-Instruct

モデル状態: 未読み込み

ページ起動時にも 0.5B を自動で読み込みます。重いと感じたら手動ボタンだけに切り替えられます。

🧠 大きいモデルは手動投入

ダウンロードしたモデルを手元で選んで読み込めます。URL は参考用です。

例: https://huggingface.co/Qwen/Qwen2.5-3B-Instruct

Qwen2.5-0.5B-Instruct などの小型モデルは上の自動取得を使ってください。
